# Student_Management_form

Designed and developed a full-stack Student Management System to digitally manage student information.
Implemented complete CRUD operations with REST APIs using Node.js and Express, integrated MongoDB for persistent storage, and added features like live search, edit/update, delete, Excel export, and automatic JSON backup.
Built a modern responsive frontend using HTML, CSS, and JavaScript to ensure smooth user interaction.
The project demonstrates practical knowledge of backend development, database integration, and real-world application architecture.
